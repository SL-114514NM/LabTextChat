# LabTextChat
使用Northwood.LabAPI以及HSM提示框架写的打字插件
# 安装
确保已经按照[HSM](https://github.com/MeowServer/HintServiceMeow)
将释放里的"LabTextChat"按照到(AppData)/SCP Secret Laboratory/LabAPI/plugins/(服务器端口)里，然后重启\
显示
```
==============
打字插件已启动
作者: 灰
聊天记录目录: xxx
==============
```
表示按照成功
# 配置文件
````
# 聊天内容显示的横坐标
y_coordinate: 100
# 显示在左侧还是右侧可填right/left
show_alignment: right
# 字体大小
text_size: 25
# 消息过期时间
message_expire_seconds: 6
````
# 命令
.bc <内容> 发送全体聊天\
.c <内容> 发送队伍聊天\
.ac <内容> 发送管理员求助
